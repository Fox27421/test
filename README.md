# Snake Game (Python + Pygame)

Простая, но полноценная игра «Змейка» на Python с графическим окном (Pygame). Подходит для запуска на Windows 11.

## Возможности

- управление стрелками или WASD;
- пауза (Space);
- перезапуск игры (R);
- выход (Esc или закрытие окна);
- подсчёт очков и рост змейки;
- столкновения со стеной и с самой собой.

## Требования

- Python 3.10+ (рекомендуется)
- Pygame 2.5+

## Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python snake_game.py
```

## Управление

- **Стрелки / WASD** — движение
- **Space** — пауза
- **R** — начать заново
- **Esc** — выход

## Структура проекта

- `snake_game.py` — основная логика игры
- `requirements.txt` — зависимости

## Примечания

Если окно не открывается, убедитесь, что установлен Pygame и используется локальная версия Python (не через WSL без GUI).
